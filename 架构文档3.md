# 基建与主界面系统 — 架构说明

## 0. 总览

基建系统（BaseBuilding）是独立于 HexMap 的局外资源转换系统，运行在 `Lvl_MainMenu` 关卡上。

### 0.1 核心循环位置

```
HexMap 探索结束 (ConcludeGame)
  → SaveGame → OpenLevel("Lvl_MainMenu")
    → 主界面：基建（制作/贸易/兑换） / 战备（食物换AP → 进入 HexMap）
```

### 0.2 面板层级

```
AMainMenuGameMode
 └── AMainMenuPlayerController
      └── UMainMenuHUDWidget                 主界面 HUD
           ├── UGamePrepPanel                战备面板
           │     └── 食物/优质口粮 → AP → InitializeNewRun → EnterRegion → OpenLevel
           └── UBaseBuildingPanel            基建面板
                ├── UCraftingPanel           制作站（始终可用）
                ├── UTradeStationPanel       贸易站（需解锁）
                └── UExchangePanel           交易所（需解锁）
```

### 0.3 类继承 — Icon 体系

```
UIcon                         Icon.h         基类图标 + OnIconClickedNative(非动态委托,lambda可用)
 └── UInfrastructureIcon       +QuantityText  带数量的图标
      └── UGoodsIcon           +PriceText     交易所商品图标，控折叠
                               +DiscountText
```

## 1. 数据层修改

### 1.1 EResourceType 新增 (DS.h)

```cpp
BuildMaterials = 7  // 建材 — 解锁贸易站/交易所
Money = 8            // 钱   — 贸易/交易所货币，不在背包显示
```

### 1.2 EResourceTypeMeta 更新

```cpp
// Money 与 AP 均不可在背包显示
inline bool IsStorable(EResourceType Type)
{
    return Type != EResourceType::AP && Type != EResourceType::Money;
}
```

### 1.3 新增 DataTable 行结构体 (DS.h)

**FInfrastructureCraftRecipe：** InputType → (OutputType, InputPerOutput)

**FInfrastructureTradeRate：** ResourceType → MoneyPerUnit

**FInfrastructureExchangeItem：** ResourceType + MoneyPerUnit + InitialStock + bIsLimited + DiscountRate

三个结构体均不使用 RowName 查找——代码通过 `GetAllRows` + 遍历匹配字段。

```cpp
bool bHasUnlockedTradeStation = false;

```

### 1.5 URougeliteRunSubsystem 新增

**委托：**

## 3. 战备系统

### 3.1 [UGamePrepPanel] (继承自 UUserWidget)

**职责：** 食物换 AP + 初始化 HexMap 运行 + 跳转。

**Properties：**
```cpp
int32 APPerFood = 1;                // 每份口粮换多少 AP（编辑器可配）
int32 APPerHighQualityFood = 3;     // 每份优质口粮换多少 AP
EHexRegionType StartingRegion = Wood; // 起始区域
```

**流程：**
```
Open()
  → RefreshOwnedDisplay()          读 RunSubsystem GlobalResource[Food/HQFood]
  → Reset + RefreshConsumeDisplay() 


+/- 按钮 → qty 变化 → RefreshConsumeDisplay() → CalculateAP()
  AP = FoodToConsume × APPerFood + HQFoodToConsume × APPerHighQualityFood


OnConfirmClicked()
  1. TryConsumeResource(Food) + TryConsumeResource(HQFood)
  2. SetGlobalResourceAmount(AP, CalculatedAP)
  3. HexMapSubsystem->InitializeNewRun(随机种子)
  4. HexMapSubsystem->EnterRegion(StartingRegion)
  5. OpenLevel("Lvl_HexMap")
```

**文件：** `Source/Arknight/BaseBuilding/GamePrepPanel.h/.cpp`

## 4. 基建面板系统

### 4.1 [UBaseBuildingPanel] (继承自 UUserWidget)

**职责：** 三个子面板的容器 + 解锁逻辑 + Money 显示。

**解锁状态表：**

| 控件 | 已解锁 | 未解锁 |
|------|:---:|:---:|
| TradeStationButton | Visible | Collapsed |
| TradeStationLockedImage | Collapsed | Visible |
| UnlockTradeStationButton | Collapsed | Visible |
| ExchangeButton | Visible | Collapsed |
| ExchangeLockedImage | Collapsed | Visible |
| UnlockExchangeButton | Collapsed | Visible |
| CraftingButton | Visible | 始终 Visible |

**解锁流程：**
```
点击透明 UnlockButton
  → GetGlobalResourceAmount(BuildMaterials)
    → 够 → TryConsumeResource + UnlockXxx() → RefreshUnlockState()
    → 不够 → 日志提示
```

**Money 刷新：** `NativeConstruct` 绑定 `OnMoneyChanged` → 自动刷新 MoneyText。

**文件：** `Source/Arknight/BaseBuilding/BaseBuildingPanel.h/.cpp`

---

### 4.2 统一面板刷新模式

三个站点面板遵循相同的"左右分离 + 对象刷新与操作分离"模式：

```
Open()
  → PopulateResourceList()            // 左侧：读资源 → 创建 Icon → 绑定 OnIconClicked
  → reset state
  → SetIcon()                         // 右侧：刷新图标显示
  → SetQuantityTexts()                // 右侧：刷新数量文本 + 按钮启用状态

OnResourceSelected(Type/Index)
  → 查表加载配方/汇率/商品
  → 重置 qty = 0
  → SetIcon() + SetQuantityTexts()

OnIncreaseClicked / OnDecreaseClicked
  → qty++/--
  → SetQuantityTexts()                // 单一真相源：限幅 + 刷新文本 + 刷新按钮

OnConfirmClicked()
  → TryConsumeResource + AddResource
  → PopulateResourceList()            // 刷新左侧
  → reset state
  → SetIcon() + SetQuantityTexts()    // 清空右侧
```

### 4.3 [UCraftingPanel] (继承自 UUserWidget)

**职责：** 消耗资源 A 制作资源 B。配方由 DT_CraftRecipe 定义。

**查表：** 遍历 `GetAllRows<FInfrastructureCraftRecipe>` → `InputType == SelectedType`

**计算公式：**
```
CostQty = CurrentProductQty × InputPerOutput
```

**按钮上限：** `MaxProduct = Owned / InputPerOutput`（整数除）

**文件：** `Source/Arknight/BaseBuilding/CraftingPanel.h/.cpp`

---

### 4.4 [UTradeStationPanel] (继承自 UUserWidget)

**职责：** 卖资源换 Money。汇率由 DT_TradeRate 定义。

**查表：** 遍历 `GetAllRows<FInfrastructureTradeRate>` → `ResourceType == SelectedType`

**计算公式：**
```
MoneyGained = CurrentSellQty × MoneyPerUnit
```

**按钮上限：** `Owned`（持有量）

**文件：** `Source/Arknight/BaseBuilding/TradeStationPanel.h/.cpp`

---

### 4.5 [UExchangePanel] (继承自 UUserWidget)

**职责：** 用 Money 买资源。商品由 DT_ExchangeItem 定义。支持限量/不限量、折扣。

**左侧 Icon：** 全部使用 UGoodsIcon，每行一个：
```cpp
GoodsIcon->SetResourceType(Item.ResourceType);
GoodsIcon->SetExchangeItem(&Item, Stock);  // 存指针+库存 → Update() 自刷新
```

**UGoodsIcon 内部显隐逻辑：**
```cpp
Update()
  if ExchangeItemRow == nullptr → SetNone() 自毁
  Super::Update()               → 图标+名称+展开QuantityText
  PriceText   → MoneyPerUnit
  QuantityText → bIsLimited → SetQuantity(Stock) / 不限量 → Collapsed
  DiscountText → DiscountRate > 0 → "-XX%" / 否则 → Collapsed
```

**计算公式（含折扣）：**
```
EffectivePrice = DiscountRate > 0 ? RoundToInt(MoneyPerUnit × (1 - DiscountRate)) : MoneyPerUnit
TotalCost = CurrentBuyQty × EffectivePrice
MaxQty = min(Money / EffectivePrice, bIsLimited ? Stock : ∞)
```

**库存刷新：** `RefreshAllStockDisplay()` 遍历 ExchangeIcons → `SetExchangeItem(&Item, Stock)`

**Money 自动刷新：** 绑定 `OnMoneyChanged` 委托 → `RefreshMoneyHoldDisplay()` + 重新 clamp 购买量。

**文件：** `Source/Arknight/BaseBuilding/ExchangePanel.h/.cpp`
