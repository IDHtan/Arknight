# 开发文档 — 基建与主界面系统

面向美术和策划。不涉及 C++ 代码编写。

---

## 一、新增资源类型

以下两个新资源类型已通过 C++ 枚举定义，需在 `DT_IconImage` 中增加对应行：

| RowName | 用途 |
|---|---|
| BuildMaterials | 建材，用于解锁贸易站和交易所 |
| Money | 钱，贸易站卖资源获得、交易所买资源消耗 |

**注意：** Money 不在背包中显示（代码已排除），仅通过基建面板的 MoneyText 查看。

## 二、新增 DataTable 配置

### DT_CraftRecipe — 制作站配方表

**行结构：** FInfrastructureCraftRecipe

| 列 | 类型 | 说明 |
|---|---|---|
| InputType | EResourceType | 消耗什么资源 |
| OutputType | EResourceType | 产出什么资源 |
| InputPerOutput | int32 | 每产出 1 个需要消耗多少输入 |

**RowName 命名任意**（代码通过 `GetAllRows` 遍历匹配，不依赖行名）。

**示例行：**

| RowName | InputType | OutputType | InputPerOutput |
|---|---|---|---|
| WoodToBuild | Wood (原木) | BuildMaterials (建材) | 5 |
| RockToBuild | Rock (原石) | BuildMaterials (建材) | 5 |

---

### DT_TradeRate — 贸易站汇率表

**行结构：** FInfrastructureTradeRate

| 列 | 类型 | 说明 |
|---|---|---|
| ResourceType | EResourceType | 卖什么资源 |
| MoneyPerUnit | int32 | 每卖 1 个获得多少 Money |

**注意：** 没有对应行的资源类型无法出售（右侧显示空白）。

---

### DT_ExchangeItem — 交易所商品表

**行结构：** FInfrastructureExchangeItem

| 列 | 类型 | 说明 |
|---|---|---|
| ResourceType | EResourceType | 可购买什么资源 |
| MoneyPerUnit | int32 | 每买 1 个消耗多少 Money（原价） |
| InitialStock | int32 | 初始库存（仅限量商品生效） |
| bIsLimited | bool | true=限量（显示余量和库存），false=无限（不显示数量） |
| DiscountRate | float | 折扣率。0=无折扣。0.3=7折。仅影响显示和实际扣款，不影响原价列 |

**左侧 Icon 自动规则：**
- `bIsLimited = true` → 显示图标 + 名称 + 售价 + 余量（0 时显示"0"）
- `bIsLimited = false` → 显示图标 + 名称 + 售价（无余量）
- `DiscountRate > 0` → 额外显示折扣标签（如"-30%"）
- 所有行都生成 Icon，包括余量为 0 的行

## 三、UMG Widget 清单与约束

以下控件名不可修改——C++ 代码通过 `BindWidget` 按名称自动绑定。

### 4.1 WBP_InfrastructureIcon（带数量的通用图标）

**继承：** WBP_Icon

在 WBP_Icon 基础上增加一个 `UTextBlock`：

| 控件名 | 控件类型 | 说明 |
|---|---|---|
| QuantityText | UTextBlock | 资源数量，代码调用 `SetQuantity(int32)` 设置 |

制作站和贸易站的左侧资源列表使用此类。

---

### 4.2 WBP_GoodsIcon（交易所商品图标）

**继承：** WBP_InfrastructureIcon

再增加两个 `UTextBlock`：

| 控件名 | 控件类型 | 说明 |
|---|---|---|
| PriceText | UTextBlock | 售价 |
| DiscountText | UTextBlock | 折扣标签（无折扣时自动折叠） |

QuantityText（继承自 InfrastructureIcon）在不限量商品上自动折叠。

---

### 4.3 WBP_MainMenuHUD（主界面画布）

| 控件名 | 控件类型 | 说明 |
|---|---|---|
| OpenBaseBuildingButton | UButton | "基建"按钮 |
| OpenGamePrepButton | UButton | "战备"按钮 |
| BaseBuildingPanel | UBaseBuildingWidget | 基建子面板引用 |
| GamePrepPanel | UGamePrepWidget | 战备子面板引用 |

两个子面板默认 `Collapsed`，代码控制显隐。

---

### 4.4 WBP_GamePrepPanel（战备面板）

| 控件名 | 控件类型 | 说明 |
|---|---|---|
| FoodIcon | UIcon (WBP_Icon) | 口粮图标（预设 bHasLinkedResourceType=Food） |
| HighQualityFoodIcon | UIcon (WBP_Icon) | 优质口粮图标（预设 bHasLinkedResourceType=HighQualityFood） |
| APIcon | UIcon (WBP_Icon) | AP 图标（预设 bHasLinkedResourceType=AP） |
| FoodOwnedText | UTextBlock | 持有口粮数 |
| HQFoodOwnedText | UTextBlock | 持有优质口粮数 |
| APGainText | UTextBlock | 将获得的 AP 数 |
| FoodConsumeText | UTextBlock | 当前选定消耗的口粮数 |
| HQFoodConsumeText | UTextBlock | 当前选定消耗的优质口粮数 |
| FoodIncreaseButton | UButton | 口粮 +1 |
| FoodDecreaseButton | UButton | 口粮 -1 |
| HQFoodIncreaseButton | UButton | 优质口粮 +1 |
| HQFoodDecreaseButton | UButton | 优质口粮 -1 |
| ConfirmButton | UButton | 确认兑换并进入 HexMap |
| ExitButton | UButton | 退出 |

**兑换规则（编辑器可配）：**
- `APPerFood`：每份口粮换多少 AP（默认 1）
- `APPerHighQualityFood`：每份优质口粮换多少 AP（默认 3）
- `StartingRegion`：起始区域（默认 Wood）

确认后自动初始化 HexMap 探索并跳转。

### 4.5 WBP_BaseBuildingPanel（基建面板）

| 控件名 | 控件类型 | 说明 |
|---|---|---|
| CraftingButton | UButton | 打开制作站 |
| TradeStationButton | UButton | 打开贸易站（解锁后可见） |
| ExchangeButton | UButton | 打开交易所（解锁后可见） |
| TradeStationLockedImage | UImage | 贸易站锁定图（未解锁可见） |
| UnlockTradeStationButton | UButton | 全透明解锁按钮（覆盖在锁定图上） |
| ExchangeLockedImage | UImage | 交易所锁定图（未解锁可见） |
| UnlockExchangeButton | UButton | 全透明解锁按钮（覆盖在锁定图上） |
| ExitButton | UButton | 退出基建面板 |
| MoneyText | UTextBlock | 持有 Money 数量 |
| CraftingPanel | UCraftingWidget | 制作站子面板引用 |
| TradeStationPanel | UTradeStationWidget | 贸易站子面板引用 |
| ExchangePanel | UExchangeWidget | 交易所子面板引用 |

**解锁条件（编辑器可配）：**
- `TradeStationUnlockCost`：解锁贸易站所需建材（默认 10）
- `ExchangeUnlockCost`：解锁交易所所需建材（默认 10）

未解锁时按钮隐藏、锁定图+透明按钮显示。点击透明按钮检查建材数并扣除。

### 4.6 WBP_CraftingPanel（制作站面板）

| 控件名 | 控件类型 | 说明 |
|---|---|---|
| IconsContainer | UWrapBox | 资源列表容器（代码动态创建） |
| CostIcon | UIcon (WBP_Icon) | 消耗的素材图标 |
| ProductIcon | UIcon (WBP_Icon) | 生成的产物图标 |
| CostQuantityText | UTextBlock | 消耗数量 |
| ProductQuantityText | UTextBlock | 生成数量 |
| IncreaseButton | UButton | 产物 +1 |
| DecreaseButton | UButton | 产物 -1 |
| ConfirmButton | UButton | 确认制作 |
| ExitButton | UButton | 退出 |

编辑器设 `IconWidgetClass = WBP_InfrastructureIcon`、`IconSize = (100, 100)`、`SlotPadding = (8, 8)`。

IconsContainer 不放子控件——代码动态创建 InfrastructureIcon。

点击左侧资源 → 查 DT_CraftRecipe → 右侧显示配方。未匹配到配方时右侧留空。

### 4.7 WBP_TradeStationPanel（贸易站面板）

| 控件名 | 控件类型 | 说明 |
|---|---|---|
| IconsContainer | UWrapBox | 资源列表容器（代码动态创建） |
| CostIcon | UIcon (WBP_Icon) | 卖出的资源图标 |
| MoneyIcon | UIcon (WBP_Icon) | 预设 bHasLinkedResourceType=Money |
| CostQuantityText | UTextBlock | 卖出数量 |
| MoneyQuantityText | UTextBlock | 获得 Money 数量 |
| IncreaseButton | UButton | 卖出 +1 |
| DecreaseButton | UButton | 卖出 -1 |
| ConfirmButton | UButton | 确认交易 |
| ExitButton | UButton | 退出 |

编辑器设 `IconWidgetClass = WBP_InfrastructureIcon`。IconsContainer 不放子控件。

点击左侧资源 → 查 DT_TradeRate → 右侧显示汇率。未匹配时不可交易。

---

### 4.8 WBP_ExchangePanel（交易所面板）

| 控件名 | 控件类型 | 说明 |
|---|---|---|
| IconsContainer | UWrapBox | 商品列表容器（代码动态创建） |
| TargetIcon | UIcon (WBP_Icon) | 要买的资源图标 |
| CostMoneyText | UTextBlock | 消耗 Money 数量 |
| GainResourceText | UTextBlock | 买得资源数量 |
| IncreaseButton | UButton | 购买 +1 |
| DecreaseButton | UButton | 购买 -1 |
| MoneyHoldText | UTextBlock | 持有 Money 数量（响应 Money 变化自动刷新） |
| ConfirmButton | UButton | 确认购买 |
| ExitButton | UButton | 退出 |

编辑器设 `GoodsIconWidgetClass = WBP_GoodsIcon`。IconsContainer 不放子控件。

折扣计算：有效价格 = 原价 × (1 - DiscountRate)。购买时按有效价格扣款。
